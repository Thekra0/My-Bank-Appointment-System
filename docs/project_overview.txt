====================================
نظرة عامة على المشروع
====================================

فكرة المشروع:
نظام ويب متكامل لإدارة وحجز مواعيد البنك، يسمح للعملاء بحجز مواعيدهم بسهولة،
وتقييم الخدمات، وعرض معلومات الفروع، مع لوحة تحكم للمدير لمتابعة الإحصائيات.

====================================
التقنيات المستخدمة:
====================================

Backend:
- PHP 7.4+ (لغة البرمجة الأساسية)
- MySQL (قاعدة البيانات)
- MySQLi (للاتصال بقاعدة البيانات)

Frontend:
- HTML5 (هيكلة الصفحات)
- CSS3 (التنسيق والتصميم)
- Bootstrap 5 (إطار عمل واجهة المستخدم)
- JavaScript (البرمجة التفاعلية)
- Chart.js (الرسوم البيانية)
- SweetAlert2 (الإشعارات)
- Font Awesome (الأيقونات)

====================================
هيكل المشروع والملفات:
====================================

1. config/db.php
   - إعدادات الاتصال بقاعدة البيانات
   - دوال مساعدة (تنظيف البيانات، التحقق من تسجيل الدخول)
   - إدارة الجلسات (Sessions)

2. auth/
   - login.php: صفحة تسجيل الدخول
   - register.php: صفحة التسجيل
   - logout.php: تسجيل الخروج

3. includes/
   - header.php: رأس الصفحة (HTML head + CSS)
   - navbar.php: شريط التنقل الديناميكي
   - footer.php: تذييل الصفحة (JS libraries)

4. pages/
   - appointments.php: إدارة المواعيد (إضافة، تعديل، إلغاء، حذف)
   - dashboard.php: لوحة التحكم الإدارية (للمدير فقط)
   - rating.php: نظام التقييمات
   - branches.php: عرض الفروع مع الخرائط

5. assets/
   - css/style.css: ملف التنسيق الرئيسي
   - js/script.js: الدوال البرمجية التفاعلية
   - images/: مجلد الصور

6. index.php
   - الصفحة الرئيسية
   - نموذج الحجز السريع
   - عرض الخدمات والمواعيد القادمة

7. database.sql
   - ملف قاعدة البيانات الكامل
   - يحتوي على الجداول والبيانات التجريبية

====================================
قاعدة البيانات:
====================================

الجداول الرئيسية:

1. users (المستخدمون)
   - id: معرف المستخدم
   - name: الاسم الكامل
   - email: البريد الإلكتروني (فريد)
   - password: كلمة المرور المشفرة
   - role: الدور (user/admin)
   - created_at: تاريخ التسجيل

2. appointments (المواعيد)
   - id: معرف الموعد
   - user_id: معرف المستخدم (Foreign Key)
   - service: نوع الخدمة
   - date: تاريخ الموعد
   - time: وقت الموعد
   - status: الحالة (upcoming/completed/cancelled)
   - created_at: تاريخ الحجز

3. ratings (التقييمات)
   - id: معرف التقييم
   - user_id: معرف المستخدم (Foreign Key)
   - appointment_id: معرف الموعد (Foreign Key)
   - rating: التقييم (1-5)
   - feedback: التعليق
   - created_at: تاريخ التقييم

4. branches (الفروع)
   - id: معرف الفرع
   - name: اسم الفرع
   - address: العنوان
   - map_embed: رابط خريطة Google Maps
   - created_at: تاريخ الإضافة

العلاقات بين الجداول:
- appointments.user_id → users.id (علاقة Many-to-One)
- ratings.user_id → users.id (علاقة Many-to-One)
- ratings.appointment_id → appointments.id (علاقة One-to-One)

====================================
المميزات الرئيسية:
====================================

1. نظام مصادقة آمن
   - تشفير كلمات المرور (password_hash)
   - حماية من SQL Injection (Prepared Statements)
   - إدارة الجلسات

2. إدارة المواعيد
   - حجز موعد جديد
   - تعديل المواعيد القادمة
   - إلغاء المواعيد
   - حذف المواعيد
   - طباعة إيصال الموعد

3. نظام التقييمات
   - تقييم الخدمات بالنجوم (1-5)
   - إضافة تعليقات
   - عرض تقييمات العملاء

4. لوحة التحكم الإدارية
   - إحصائيات شاملة
   - رسوم بيانية تفاعلية (Chart.js)
   - عرض أحدث المواعيد

5. عرض الفروع
   - معلومات تفصيلية عن كل فرع
   - خرائط Google Maps التفاعلية
   - معلومات الاتصال وأوقات العمل
